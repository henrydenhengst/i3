# ❄️ Nordic i3wm Cheat Sheet

Dit is het overzicht van jouw specifieke configuratie voor de oude laptop, geoptimaliseerd voor een workflow met **Kitty** en **Falkon**.

---

## 🚀 Snelkoppelingen (Basis)
| Toetsencombinatie | Actie |
| :--- | :--- |
| **`$mod + Enter`** | Open **Kitty** (Terminal) |
| **`$mod + w`** | Open **Falkon** (Browser) |
| **`$mod + d`** | Open **Rofi** (Programma zoeken/starten) |
| **`$mod + Shift + q`** | Sluit het actieve venster |
| **`$mod + f`** | Wissel tussen Fullscreen en Tegel-modus |

---

## 🧭 Navigatie & Workspaces
| Toetsencombinatie | Actie |
| :--- | :--- |
| **`$mod + 1`** | Ga naar **Workspace 1** (Browser 󰖟) |
| **`$mod + 2`** | Ga naar **Workspace 2** (Terminal 󰆍) |
| **`$mod + j / k / l / ;`** | Focus verplaatsen (Vim-stijl: Links, Onder, Boven, Rechts) |
| **`$mod + Shift + [1-4]`** | Verplaats het huidige venster naar workspace 1, 2, 3 of 4 |

---

## 🛠️ Systeembeheer
| Toetsencombinatie | Actie |
| :--- | :--- |
| **`$mod + Shift + r`** | **Herstart i3** (Vernieuwt config & kiest nieuwe wallpaper) |
| **`$mod + Shift + e`** | Afmelden (Terug naar DMLight/Login scherm) |
| **`$mod + minus (-)`** | Toon/verberg de **Scratchpad** (Zwevend venster) |
| **`$mod + Shift + -`** | Verplaats venster naar de Scratchpad |

---

## 💻 Laptop Hardware
| Toets / Combinatie | Actie |
| :--- | :--- |
| **Volume Up / Down** | Geluid harder of zachter (stappen van 5%) |
| **Brightness Up / Down**| Schermhelderheid aanpassen |
| **`$mod + r`** | Resize-modus (gebruik pijltjestoetsen, daarna `Esc`) |

---

## 📂 Belangrijke Paden
* **i3 Config:** `~/.config/i3/config`
* **Kitty Config:** `~/.config/kitty/kitty.conf`
* **Wallpapers:** `/usr/share/backgrounds/`
* **Polybar:** `~/.config/polybar/config.ini`
